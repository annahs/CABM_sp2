# CABM_sp2

Parses raw data and creates time series of refractory black carbon mass and number concentration from sp2 measurements made at the ECCC CABM sites.


## Requirements

* Python >= 2.7  
* NumPy >= 1.11
* matplotlib >= 1.5
* SciPy >= 0.18
* python-dateutil >= 2.6
* mysql-connector-python >= 2.1
* MySQL >= 5.7
